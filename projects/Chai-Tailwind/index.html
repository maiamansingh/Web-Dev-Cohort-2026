How It Works

Understanding the Vanilla JavaScript Engine

1. The Dictionary Map

Instead of using complex Regular Expressions to parse strings, ChaiTailwind uses a highly reliable dictionary object. Every supported class name (e.g., chai-bg-red) is mapped directly to its exact CSS property and value.

2. DOM Traversal

When the page loads, a simple beginner-friendly script grabs every single HTML element using getElementsByTagName("*"). It then loops through every element to check its class list.

3. Inline Application

If the script finds a class that matches a key in our dictionary, it extracts the target property (like backgroundColor) and applies the value directly to the element's inline style object.

Example Output:

// Input HTML
<div class="chai-p-2">

// Engine Output
<div style="padding: 16px;">
