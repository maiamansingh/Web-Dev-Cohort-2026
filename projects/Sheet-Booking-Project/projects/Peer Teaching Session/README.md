## 🛠️ Project Overview
This session focuses on making HTML easy to understand, focusing on structure, accessibility, and modern best practices.

### Key Topics Covered:
* **Semantic HTML:** Using `<header>`, `<main>`, and `<footer>` correctly.
* **Forms & Inputs:** Building user-friendly data entry points.
* **Metadata:** Understanding the `<head>` section and SEO basics.



## 💻 Tech Stack
* **Language:** HTML5
* **Framework:** Tailwind CSS (for styling)
* **Recording:** OBS Studio / Zoom
